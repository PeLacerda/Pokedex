# Pokedex

Uma Pokédex Pokémon interativa desenvolvida para facilitar a busca de informações sobre os Pokémon. Com esta aplicação, você pode pesquisar um Pokémon pelo seu nome ou número e visualizar rapidamente detalhes como imagem, nome, número e tipo(s).

## Funcionalidades

- Buscar Pokémon pelo nome ou número.
- Exibir imagem oficial do Pokémon.
- Exibir nome, número e tipos do Pokémon encontrado.
- Interface simples e intuitiva.

## Como usar

1. Clone o repositório:
   ```sh
   git clone https://github.com/PeLacerda/Pokedex.git
   ```

2. Acesse o diretório do projeto:
   ```sh
   cd Pokedex
   ```

3. Instale as dependências (ajuste este passo conforme o gerenciador de pacotes utilizado, ex: npm, yarn, pip, etc.):
   ```sh
   npm install
   # ou
   yarn install
   ```

4. Inicie o projeto:
   ```sh
   npm start
   # ou
   yarn start
   ```

5. Abra o navegador e acesse `http://localhost:3000` (ou a porta configurada).

## Tecnologias Utilizadas

- HTML, CSS e JS.
- PokeAPI
